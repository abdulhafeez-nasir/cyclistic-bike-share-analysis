# Cyclistic Bike-Share Case Study

## Overview
Analysis of behavioral differences between annual members and casual riders to support membership growth strategy.

## Business Question
How do annual members and casual riders use Cyclistic bikes differently?

## Tools & Methods
- SQL (Google BigQuery)
- Tableau
- Data cleaning and aggregation
- Exploratory data analysis

## Key Insights
- Members account for ~88% of rides and ride mostly on weekdays
- Casual riders take longer trips and peak on weekends
- Clear commuter vs leisure usage patterns

## Recommendations
- Weekend-focused membership campaigns
- Incentives for long-duration casual riders
- Improved commuter bike availability
